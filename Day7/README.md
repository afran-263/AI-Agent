# HRMS Agent Local

HRMS Agent API is an ASP.NET Core Web API that provides employee, attendance, and task management capabilities. It supports creating, retrieving, updating, and managing HR-related data through REST endpoints. The application stores and retrieves data locally, making it suitable for development, testing, demonstrations, and AI agent integrations.

---

# HRMS Attendance Agent Voyon

This API integrates with Voyon HRMS to retrieve and update employee attendance information. It allows users to:

* Get team attendance reports for a specific date.
* View present and absent employees.
* Fetch individual employee attendance details.
* Mark employee check-in and check-out attendance in Voyon HRMS.
* Read attendance data from Voyon and write attendance updates back to the system in real time.

The service acts as a bridge between external applications and Voyon HRMS, ensuring attendance records remain synchronized and up to date.
