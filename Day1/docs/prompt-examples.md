# Prompt Examples – HRMS AI Agent

This document contains a set of **initial prompt examples** that can be used to test or guide an HRMS AI Agent. These prompts simulate real-world HR, IT, manager, and employee interactions.

---

## 1. Employee Onboarding Prompts

* Create a new employee profile for John Doe in the system.
 * Generate employee credentials for a new hire in the Engineering team.
 * Start onboarding process for an employee joining as a Software Engineer.
 * What are the next onboarding steps for a new employee?
 * Show onboarding status of employee ID EMP1023.

---

## 2. Employee Self-Service Prompts

* What is my current onboarding status?
 * I want to upload my documents for verification.
 * Update my profile with my education and skills.
 * Show my pending onboarding tasks.
 * What documents are required for onboarding?

---

## 3. HR Task Management Prompts

* Assign onboarding tasks to the IT team for a new employee.
 * Create a task to verify employee documents.
 * Show all pending HR onboarding tasks.
 * Which employees have incomplete onboarding?
 * Send reminders for overdue onboarding tasks.

---

## 4. Manager Related Prompts

* Show onboarding progress of my team members.
 * Assign project onboarding tasks to new employees.
 * Which employees are ready for project allocation?
 * Send me a summary of pending onboarding tasks.
 * Approve onboarding completion for employee EMP1023.

---

## 5. IT Support Prompts

* Create a company email for the new employee.
 * Set up system access for EMP1023.
 * What IT onboarding tasks are pending?
 * Mark laptop setup as completed for the new joiner.
 * Generate access credentials for internal tools.

---

## 6. AI HR Assistant (RAG-based) Prompts

* What is the company leave policy?
 * How many working hours are required per week?
 * Explain the payroll process.
 * What is the IT support request procedure?
 * Where can I find the travel reimbursement policy?

---

## 7. Skill and ATS Prompts

* Extract skills from this resume.
 * Rank candidates based on Python and React skills.
 * Suggest employees suitable for a frontend project.
 * Show top candidates for a backend developer role.
 * Match employees with project requirements.

---

## 8. Notification and Workflow Prompts

* Send reminders for pending onboarding tasks.
 * Notify HR about incomplete document submissions.
 * Alert manager about overdue approvals.
 * Show all unread notifications for employee EMP1023.
 * Escalate delayed IT setup tasks.

---

## 9. System-Level / Agent Control Prompts

* What tasks are currently in progress?
 * Show full onboarding workflow for all employees.
 * Summarize system-wide onboarding status.
 * Which department has the most pending tasks?
 * Log all recent onboarding activities.

---

## Summary

These prompts help simulate real-world usage of an HRMS AI Agent across HR operations, onboarding workflows, IT provisioning, and employee self-service scenarios.
